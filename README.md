 OpenCart Automation Testing Framework

## 📌 Project Overview
This project is a **Selenium-based Automation Testing Framework** developed for testing an OpenCart E-Commerce web application.  
It is designed using **Hybrid Framework (Page Object Model + Cucumber BDD)** to ensure scalability, reusability, and maintainability.

The framework covers end-to-end testing of core e-commerce functionalities like product search, add to cart, and checkout process.
